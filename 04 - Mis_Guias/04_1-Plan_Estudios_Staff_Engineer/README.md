# 🎓 Plan de Estudios Staff Engineer — Zero to Hero v2

Bienvenido al repositorio central de tu preparación técnica avanzada. Este plan de estudios está diseñado específicamente para llevar tu perfil de **Senior Software Engineer** al nivel de **Staff Engineer / Software Architect**, con un enfoque profundo en fundamentos, diseño de sistemas a escala y las nuevas fronteras de la IA.

- **[00-0-README (Plan Maestro)](./00-0-README.md)**: La filosofía, principios y reglas del sistema.
- **[00-1-Diagnóstico Inicial](./00-1-diagnostico.md)**: Evaluación para identificar tus gaps de conocimiento reales.
- **[00-2-Roadmap Visual](./00-2-roadmap-visual.md)**: El mapa global de dependencias y tiempos estimados.

## 📑 Índice de Módulos

### [Módulo 1: CS Fundamentals](./01-00-overview.md)
*La base de todo el conocimiento avanzado.*
- Estructuras de datos, complejidad algoritmica, gestión de memoria, sistemas operativos, concurrencia base, redes/protocolos y fundamentos de bases de datos.

### [Módulo 2: Algoritmos y Patrones de Coding](./02-00-overview.md)
*Dominio de la resolución de problemas para entrevistas y optimización.*
- Patrones lineales y no lineales, Dynamic Programming, Grafos avanzados, C# específico para DSA y simulación de entrevistas.

### [Módulo 3: Arquitectura y Diseño de Software](./03-00-overview.md)
*Diseño de código mantenible, flexible y escalable a nivel de aplicación.*
- SOLID, Patrones GoF, Clean Architecture, DDD (Domain-Driven Design), CQRS, Event Sourcing, Diseño de APIs y Estrategias de Testing.

### [Módulo 4: System Design a Escala](./04-00-overview.md)
*El corazón del rol de Staff/Arquitecto.*
- Caching, Message Queues, Sistemas Distribuidos, Observabilidad, Seguridad, Casos Clásicos (Twitter, Uber, etc.) y CI/CD de alta disponibilidad.

### [Módulo 5: Stack Tecnológico Avanzado](./05-00-overview.md)
*Profundización en las herramientas del ecosistema moderno.*
- .NET avanzado, Azure primario, suficiencia en Python y TypeScript, y Arquitectura Frontend con React.

### [Módulo 6: IA Integrada en el Ciclo de Vida](./06-00-overview.md)
*Preparación para la ingeniería del futuro (2026+).*
- LLMs en System Design, Agentes y Orquestación, Context Engineering y Evaluación de Seguridad en modelos.

### [Módulo 7: Carrera y Entrevistas Staff](./07-00-overview.md)
*Conversión de conocimiento en ofertas reales.*
- Estrategia de entrevistas de Coding, System Design (Tradicional e IA), Entrevistas Behavioral (STARL), estrategia por empresa y Negociación.
