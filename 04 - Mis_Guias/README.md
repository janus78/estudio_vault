# 🧠 Mis Guías Técnicas

Este directorio contiene documentación profunda y frameworks de trabajo creados para la transición de **Senior Developer** a **Staff Engineer / Architect**. 

Las guías se han organizado en subcarpetas temáticas para facilitar su mantenimiento y consulta.

## 📂 Categorías de Guías

- **[01 - Guías Base IA](./01-Guias_Base_IA/)**: Sistema de trabajo con IA, optimización de modelos y gestión de suscripciones.
- **[02 - Comunicación](./02-Comunicacion/)**: Estrategias de comunicación de valor e influencia para ingenieros senior.
- **[03 - DotNet & Arquitectura](./03-DotNet_Recursos/)**: Material técnico sobre .NET, System Design, Frontend y Bases de Datos.
- **[04.1 - Plan de Estudios (Staff)](./04_1-Plan_Estudios_Staff_Engineer/)**: Módulos detallados del plan de estudios CS y algoritmos.
- **[05 - Varios](./05-Varios/)**: Roadmaps generales, guías de NotebookLM y herramientas especializadas.

---
_Nota: Estas guías son documentos vivos que se actualizan conforme avanza el proceso de aprendizaje y se descubren nuevos patrones en producción._
