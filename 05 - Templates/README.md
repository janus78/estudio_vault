# Templates

Directorio para plantillas reutilizables de notas, ejercicios, documentos o estructuras base.

## 🤖 Prompts de Inteligencia Artificial

Estas plantillas están diseñadas para gestionar el contexto y la continuidad de las sesiones de trabajo con asistentes de IA (Claude, ChatGPT, etc.).

- **[001-Prompt_Claude_Project_ContextFile](./001-Prompt_Claude_Project_ContextFile.md)**: Genera un documento de contexto al finalizar una sesión para "guardar" el progreso.
- **[002-Prompt_Claude_Projects_Arranque_Usa_Conttexto](./002-Prompt_Claude_Projects_Arranque_Usa_Conttexto.md)**: Reinicia una sesión de trabajo utilizando un archivo de contexto previo.
- **[003-Prompt_Generico_Obtener_Resumen_Contexto](./003-Prompt_Generico_Obtener_Resumen_Contexto.md)**: Versión genérica para extraer resúmenes de contexto de cualquier chat.
- **[004-Prompt_Generico_Arranque_Resumen_Contexto](./004-Prompt_Generico_Arranque_Resumen_Contexto.md)**: Versión genérica para arrancar una sesión basada en un resumen previo.
