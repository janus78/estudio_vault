# 01-Router Multi IA

## Descripción

Este módulo forma parte del proyecto **Estudio Vault** y está diseñado para gestionar y orquestar múltiples Inteligencias Artificiales (IA) a través de un sistema de enrutamiento inteligente.

## Objetivo

Proporcionar una capa de abstracción que permita:
- Conectar diversos proveedores de IA (OpenAI, Anthropic, Google, etc.)
- Enrutar solicitudes al modelo más adecuado según el contexto
- Gestionar fallbacks y redundancia entre servicios
- Optimizar costos y latencia en las llamadas a modelos
