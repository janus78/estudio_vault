# 🏛️ System Design & Arquitectura

Material enfocado en el diseño de sistemas distribuidos, escalabilidad, disponibilidad y patrones arquitectónicos avanzados.

## 🗝️ Temas de Enfoque
- **Escalabilidad y Performance**: Load balancing, Caching, Content Delivery Networks.
- **Data Management**: Bases de datos SQL vs NoSQL, Consistencia, Replicación y Particionamiento.
- **Comunicación**: REST, gRPC, Message Queues y Event-Driven Architectures.
- **Reliability & Observability**: Logs, Métricas, Tracing y SRE principles.
