# Estudio Vault

Este directorio agrupa repositorios y materiales de apoyo para un proceso de estudio personal.

## Estructura inicial

- `00 - Inbox`: espacio para capturas rápidas, ideas o material por clasificar.
- `01 - Conceptos`: notas y referencias sobre conceptos de estudio.
- `02 - Entrevistas`: preparación, preguntas y material relacionado con entrevistas.
- `03 - System_Design`: contenido de estudio sobre diseño de sistemas.
- `04 - Mis_Guias`: guías propias, resúmenes o documentación personal.
- `05 - Templates`: plantillas reutilizables para notas, ejercicios o repositorios.

Cada carpeta tiene su propio `README.md` inicial para facilitar su conversión posterior en repositorio Git.
