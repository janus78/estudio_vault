# 🏛️ Estudio Vault — Zero to Hero v2

Bienvenido a **Estudio Vault**, un ecosistema de conocimiento personal centralizado, diseñado y optimizado para la transición de **Senior Software Engineer** hacia roles de **Staff Engineer** y **Software Architect**. 

Este repositorio funciona como un "Segundo Cerebro" en **Obsidian**, integrando fundamentos de Computer Science, diseño de sistemas a escala y las últimas metodologías de ingeniería asistida por IA.

---

## 📂 Estructura del Vault

El contenido se organiza de forma semántica para facilitar la navegación y el aprendizaje incremental:

- **[00 - Inbox](./00%20-%20Inbox/)**: Captura rápida de ideas, recursos y material pendiente de procesar.
- **[01 - Conceptos](./01%20-%20Conceptos/)**: Enciclopedia técnica de conceptos teóricos de ingeniería.
- **[02 - Entrevistas](./02%20-%20Entrevistas/)**: Estrategias de Coding, System Design y Behavioral para niveles Staff.
- **[03 - System Design](./03%20-%20System_Design/)**: Patrones de escalabilidad, disponibilidad y arquitectura distribuida.
- **[04 - Mis Guías](./04%20-%20Mis_Guias/)**: El corazón del vault, incluyendo el **Plan de Estudios Zero to Hero**.
- **[05 - Templates](./05%20-%20Templates/)**: Plantillas para notas técnicas y prompts avanzados de IA.
- **[06 - Proyectos](./06%20-%20Proyectos/)**: Implementaciones reales como el *Router Multi-IA* para aplicar la teoría.

## 🚀 Pilares del Conocimiento (Update 2026)

### 🎓 Staff Engineer Path (Zero to Hero v2)
Un currículum estructurado de **7 módulos** diseñado para cubrir todos los gaps técnicos:
1. **CS Fundamentals:** Estructuras, Memoria y Redes.
2. **Algoritmos Avanzados:** Patrones de coding y optimización.
3. **Software Architecture:** SOLID, DDD, CQRS y Clean Arch.
4. **System Design a Escala:** Sistemas distribuidos y resiliencia.
5. **Stack Moderno:** Profundización en .NET, Azure, React y Python.
6. **IA Integrada:** Ingeniería de agentes, LLMs y orquestación.
7. **Carrera:** Estrategias de negociación y procesos Staff.

### 🤖 IA-Aided Engineering
Metodologías para elevar la productividad al nivel de "Desarrollador Aumentado":
- **Framework IA Personal:** Integración de LLMs en el flujo diario.
- **Context Engineering:** Optimización de la memoria y contexto para agentes.
- **NotebookLM:** Investigación técnica acelerada.

### 🏛️ Arquitectura y Producción
Foco en decisiones técnicas que impactan al negocio:
- **Observabilidad:** Tracing, métricas y seguridad proactiva.
- **Bases de Datos:** Fundamentos de producción y gestión de estado distribuido.
- **Comunicación:** Estrategias de influencia técnica y liderazgo.
